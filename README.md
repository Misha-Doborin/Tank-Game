# Tank-Game

Игра на Python/Pygame. Проект разделён на модули, чтобы код было проще развивать как настоящий игровой проект.

## Установка зависимостей

```bash
python -m pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Структура проекта

- `main.py` — минимальная точка входа: импортирует `Game` и запускает игру.
- `requirements.txt` — список Python-зависимостей проекта.
- `tank_game/constants.py` — размеры экрана/мира, цвета, базовые характеристики танков, таймеры и кулдауны.
- `tank_game/controls.py` — клавиши движения и соответствующие векторы направления.
- `tank_game/powerups.py` — описание усилений: название, цвет и длительность.
- `tank_game/models.py` — небольшие dataclass-модели: пули, усиления, канал усиления и частицы эффектов.
- `tank_game/common.py` — общий набор импортов, который используют игровые модули.
- `tank_game/entities/ghost.py` — логика и отрисовка призрака и формы гиганта.
- `tank_game/entities/tank.py` — логика танка: характеристики, таймеры, урон, скорость и отрисовка.
- `tank_game/entities/__init__.py` — публичные экспорты сущностей `Ghost` и `Tank`.
- `tank_game/game/core.py` — основной класс `Game`: инициализация, главный цикл, события, общий update и сообщения.
- `tank_game/game/spawn.py` — создание препятствий, стартовых танков и точек респавна.
- `tank_game/game/ghost_actions.py` — действия призрака: вселение, выселение, щупальце, гигант, предатель и стрельба гиганта.
- `tank_game/game/ai.py` — AI танков: выбор цели, уклонение, движение к усилениям и боевое поведение.
- `tank_game/game/movement.py` — движение, столкновения, каменная стена, проверка линии огня и камера.
- `tank_game/game/combat.py` — стрельба, ракеты, желчь, попадания, взрывы и респавн после уничтожения.
- `tank_game/game/powerups.py` — появление, сбор, применение и притягивание усилений.
- `tank_game/game/render.py` — отрисовка мира, HUD, миникарты, пуль, частиц и канала усилений.
- `tank_game/game/__init__.py` — публичный экспорт класса `Game`.
- `tank_game/__init__.py` — публичный экспорт пакета игры.
